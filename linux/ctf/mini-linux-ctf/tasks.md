# Mini Linux CTF – Tasks

## Task 1 – Navigation
Find the first flag somewhere in the filesystem.
You may not use GUI tools.

## Task 2 – Permissions
A file exists but cannot be read.
Diagnose why and fix the permissions.

## Task 3 – Pipes
A log file contains useful information.
Extract the flag using pipes only.

## Task 4 – Execution
A script exists but will not run.
Explain why, then fix it.
