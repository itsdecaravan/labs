# Mini Linux CTF 

A small capture-the-flag style lab focused on core Linux skills.

## Focus Areas
- Navigation
- Files & permissions
- Pipes and redirection

## Rules
- Terminal only
- No GUI tools
- Explain permissions verbally

## Prerequisites
- Linux VM (recommended)
- SSH access
- Snapshot taken before starting

## Setup
Run the setup script:

./setup_ctf.sh
